# pdex-ts

## Description
A CLI Pokedex written in TypeScript.  
Explore locations, catch Pokemon, and view their information!  

### Usage:

exit: Exits the pokedex  
help: Displays a help message  
map: Displays the next 20 locations  
mapb: Displays the previous 20 locations  
explore: Displays all Pokemon found in an area  
catch: Attempts to catch specified Pokemon  
inspect: Displays information of specified Pokemon  
pokedex: Displays the name of all caught Pokemon
